Password Generator & Vault 🔐

A full-stack password management web application designed to help users create strong, secure passwords and store them safely in a personal vault.

Features include - 
Generate highly secure passwords,
Custom password length selection,
Include uppercase, lowercase, numbers and symbols,
Password strength indicator,
Copy password with one click,
User authentication (Register & Login),
Secure password vault,
Modern cybersecurity-themed UI,
Responsive design for desktop and mobile,

Tech Stack -

Frontend: HTML, Tailwind CSS, JavaScript

Backend: Python, Flask

Database: SQLite

Authentication: Flask Sessions

Security: Password hashing and secure storage

Goal -
This project was built to strengthen Python, Flask, frontend development, authentication, database management, and cybersecurity fundamentals while creating a practical real-world application.
